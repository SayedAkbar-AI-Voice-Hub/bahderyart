# Bahadery Art Portfolio

A minimalist, high-end artist portfolio website featuring galleries, a store, making-of videos, and professional biography for **Nangialai Bahadery** - Afghan fine arts artist and UAE Golden Visa holder.

## Features

- **Full-Screen Homepage**: Elegant slideshow of artwork with smooth transitions
- **Artwork Galleries**: Separate galleries for original landscapes and indoor/mural pieces
- **Store Section**: Display artwork available for purchase
- **About Page**: Professional biography with exhibitions and press coverage
- **Contact Form**: Direct communication with social media links
- **Making Of**: Behind-the-scenes videos of the creative process
- **Portfolio Manager**: Built-in admin interface to upload and manage all content

## Technology Stack

- React 19 with TypeScript
- React Router for navigation
- Vite for fast development and building
- TailwindCSS for styling
- LocalStorage for image management

## Run Locally

**Prerequisites:** Node.js (v18 or higher)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown in the terminal (typically `http://localhost:5173`)

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## Managing Content

1. Click the **"Website Setup"** button in the bottom-left corner
2. Upload your artwork images and manage metadata
3. Switch to the "Site Content" tab to update About/Contact page images
4. Click "Save & Exit" when done

All images are stored in your browser's LocalStorage.

## Deployment

This site can be deployed to:
- **GitHub Pages** (recommended)
- Netlify
- Vercel
- Any static hosting service

For GitHub Pages:
1. Build the project: `npm run build`
2. Deploy the `dist` folder to your gh-pages branch
